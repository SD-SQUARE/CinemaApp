## Task Summary

- Build Movie List page with search by movie title.
- list movies by latest.
- for vendor allow him to delete or edit the movie by click on the 3 dots.
- on click move to view movie.

## Details

- **Objective:** list movies and search them.  
  
- **Specifications:**
  - use rpc from supabase with defined function "delete_movie" to delete movie.
  - use channel from supabase to listen to changes.
  
- **Steps:**
  - on init get all movies.
  - on search fetch the movies that matches the search values.
